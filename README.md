# Runebound

Runebound is a top-down pixel art game built with the Godot engine. 

All visual assets and sprites were custom-made using Aseprite. The game features the **m5x7** font by Daniel Linssen, which is available [here](https://managore.itch.io/m5x7).

![screenshot](https://github.com/dalu-wins/runebound-godot/blob/main/screenshots/Screenshot%202026-07-06%20163618.png)

## Core Features

### Player & Systems
* **Player Animation:** Player has idle and walk animations for vertical and horizontal directions.
* **Health Mechanics:** Health tracking with an HP bar.
* **Inventory System:** Inventory with item stacking.

### Items & Interactivity
* **Consumables:** Each fruit has a different amount of health it restores.
* **Healing Preview:** UI preview showing exactly how much health an item will restore before consuming it.

### World & Map
* **Tilemap System:** Features a reduced tileset. Also contains variants.
* **Terrain Painting:** Support for Godot's terrain auto-tiling.
